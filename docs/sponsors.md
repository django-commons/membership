# Sponsors

These organizations offer their services and support to Django Commons free of charge. We are very grateful for their support. 

## 1Password

[1Password](https://1password.com/) provides the Django Commons admin team with free access to their password management service. This allows us to securely manage and share credentials for our various services and tools we need to use to run Django Commons.

## deSEC

[deSEC](https://desec.io/) is a free DNS hosting service with security in mind. They provide Django Commons with DNS hosting and sponsor domain registration.

## GitHub

GitHub provides Django Commons with increased GitHub Actions quotas free of charge.
