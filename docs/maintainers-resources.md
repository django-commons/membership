Free Resources for django-commons Maintainers
=============================================

This page contains resources for django-commons maintainers. If you know of additional resources available to
maintainers that are not listed here, please create a pull request to add them!

## AWS Open Source Credits

AWS provides free credits for open source maintainers. To apply for credits, visit [this post][aws] and check out the
section `How do I apply?`

## thanks.dev

[thanks.dev][thanks.dev] is a platform that allows open source maintainers to receive donations from enterprises. Your
project might
already be donations, register with your GitHub account.

## JetBrains IDEs Open Source License

JetBrains provides free licenses for their IDEs to open source maintainers. This includes PyCharm, which is a popular
IDE for Django development. To apply for a free license, visit [JetBrains Open Source License page][jetbrains].

## GitHub Actions Manager for JetBrains IDEs

JetBrains IDEs have a GitHub Actions Manager Plugin that allows you to manage your GitHub Actions workflows directly
from your IDE. The plugin is developed by django-commons member @cunla.

If you want a license for the plugin, create an issue in the [GitHub Actions Manager repository][ghactions-manager] with
your email and the project you are maintaining in django-commons.
